G R A P H I K    F O N T    R E G U L A R

- - - - - - - - - - - - - - - - - - - - -


FOR DESIGNERS (PRINT & DIGITAL)

- use the .OTF file
- listed in the font book as 'Graphik'
- language support for Latin, Cyrillic & Greek


FOR WEB DEVELOPERS

- use the .WOFF2, .WOFF, .EOT and .TTF files 
- load them on the website via @font-face
- language support for Latin


FOR APP DEVELOPERS

- use the .TTF file
- fonts can be installed in iOS/Android/etc apps
- language support for Latin
